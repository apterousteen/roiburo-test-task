# Тестовое ROIburo
Страница сайта с картой, 2 видами слайдеров и таблицей 

Live: https://roiburo-test-task.netlify.app/

**Время выполнения:** 11 часов

**Node:** v18.17.1

## Инструкция

Установка зависимостей

```shell
npm i
```

Запуск проекта

```shell
npm run dev
```
